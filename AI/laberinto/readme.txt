La práctica se corre usando ant. Todo el código está en Main.java

Para modificar el tamaño de la cuadrícula hay que cambiar los valores de las
variables globales gridWidth, gridHeight, y CS (ésta última para el tamaño de cada
celda).

Es posible que sea también conveniente modificar el tamaño del canvas, para que
quepa una cuadrícula muy grande. Para ello se modifican las variables globales
alto, ancho y celda.

Por último, el algoritmo corre paso a paso según el ritmo del usuario. La tecla
ENTER (o RETURN en Mac OS) es la que ejecuta un paso del algoritmo.
